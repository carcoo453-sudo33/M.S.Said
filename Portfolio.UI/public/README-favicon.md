# Favicon Setup

This folder contains the favicon files for the Mostafa.Dev portfolio website.

## Files

- `favicon.svg` - Main SVG favicon (modern browsers)
- `favicon.ico` - Fallback ICO favicon (older browsers)
- `favicon-16x16.svg` - 16x16 SVG favicon
- `favicon-32x32.svg` - 32x32 SVG favicon
- `apple-touch-icon.svg` - Apple touch icon for iOS devices
- `site.webmanifest` - Web app manifest for PWA support

## Design

The favicon uses the same "M" logo design as the site's navbar:
- Red background (#dc2626)
- White "M" letter
- Rounded corners
- Inter font family

## Browser Support

- Modern browsers: SVG favicons
- Older browsers: ICO fallback
- iOS devices: Apple touch icon
- PWA support: Web manifest

## Updating

To update the favicon design:
1. Modify the SVG files in this folder
2. Use [favicon.io](https://favicon.io/favicon-converter/) to generate a new favicon.ico
3. Update the colors/design to match the site's branding
