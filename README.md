# Portfolio Application

A modern, full-stack portfolio application built with architectural excellence and professional-grade security.

## 🚀 Overview

This project showcases a high-performance portfolio system featuring:
- **Frontend**: Angular 19 with Standalone Components and Tailwind CSS.
- **Backend**: .NET 9 Web API following Clean Architecture principles.
- **Data**: Entity Framework Core with robust migration strategies.
- **Security**: Secure file uploads, magic byte validation, and strict CORS policies.
